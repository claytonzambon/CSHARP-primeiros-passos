# 54.1.Vetores
##### Fazer um programa para ler um número inteiro positivo N, depois ler N números quaisquer e armazená-los em um vetor. Em seguida, mostrar na tela todos elementos do vetor

Exemplos:

| Entrada:  | Saída:    |
|-----------|-----------|
| 4         | 10.5      |
| 10.5      | 4.2       |
| 4.2       | -7.1      |
| -7.1      | 15.0      |
| 15.0      |           |
